# Focus and Productivity

## Deep Work
Deep work is focused, distraction-free work on cognitively demanding tasks.

Requirements:
- No phone
- Single task
- Clear goal

---

## Distraction Control
Common distractions:
- Phone notifications
- Multitasking
- Background noise

Countermeasures:
- Phone out of reach
- Website blockers
- Fixed study location
- Noise-cancelling headphones
- Scheduled breaks
- Mindfulness techniques
- Clear workspace
- Prioritize tasks
- Time blocking
- Use focus apps
- Inform others of focus times
- Set specific goals
- Limit social media use
- Practice deep breathing




- Use white noise or instrumental music
- Establish routines
- Reflect on distraction patterns
- Reward focused sessions
- Use visual cues (e.g., "Do Not Disturb" sign)
- Track focus time
- Adjust environment lighting
- Maintain healthy lifestyle (sleep, nutrition, exercise)
- Use task management tools
- Set deadlines
- Practice single-tasking
