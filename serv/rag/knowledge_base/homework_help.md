# Homework Help Principles

## How to Approach Any Homework Problem
1. Read the problem carefully
2. Identify what is asked
3. List known information
4. Choose a strategy
5. Solve step-by-step
6. Verify result

---

## When Stuck
If stuck for more than 20 minutes:
- Re-read the problem
- Solve a simpler version
- Review relevant theory
- Ask a specific question

---
