# Note Taking

## Cornell Method
Notes divided into:
- Main notes
- Cues/questions
- Summary

Encourages active review.

---

## Zettelkasten
Knowledge system based on linked atomic notes.

Rules:
- One idea per note
- Link ideas explicitly
- Write in your own words
Benefits:
- Better understanding
- Easy retrieval
- Encourages connections
Tools:
- Zettlr
- Obsidian
---
## Mind Mapping
Visual diagrams connecting ideas around a central concept.
Steps:
1. Start with main idea
2. Branch out with subtopics
3. Use colors, images, symbols
Benefits:
- Visual organization
- Stimulates creativity
- Easy overview of relationships
Tools:
- MindMeister
- XMind
---
## Outlining
Hierarchical structure of main topics and subpoints.
Steps:
1. Identify main topics
2. Add subpoints under each
Benefits:
- Clear structure
- Easy to follow
Tools:
- Microsoft Word
- Google Docs
---
## Annotation
Active engagement with text by highlighting and adding notes.
Best practices:
- Highlight key points
- Write margin notes
- Summarize sections
Benefits:
- Improved comprehension
- Better retention
Tools:
- Adobe Acrobat
- Hypothesis
